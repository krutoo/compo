What?

Almost like React but without JSX. Write markup with native tagged template literals syntax instead. 

import { html, render } from 'packmar';

function Heading ({ text, onClick }) {
    return html`<h1 onclick=${onClick}>${text}</h1>`;
}

render(Heading({ text: 'Hello, world' }), document.body);

Use?

Install with npm:

npm install --save packmar

Now seriously

This is a fully functional library for building user interface. But it was developed for training purposes in order to study frontend development: coding, unit testing, build configuration, deployment, etc.
